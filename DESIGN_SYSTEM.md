# National Park Passes - Design System Reference

This repository uses a custom design system inspired by Radix UI and modern SaaS aesthetics.

## Design Tokens
- Primary colour: `#1F3A2E`
- Secondary neutral: `#6B7280`
- Background: `#FFFFFF`
- Border: `#E5E7EB`
- Text: `#111111`
- Radius: `rounded-lg` (`12px`)

## Typography
- H1: `32px`, **bold**
- H2: `24px`, **semi-bold**
- Body: `16px`, normal

## Spacing
Apply consistent spacing:
- Section gap: `48px`
- Form field gap: `24px`
- Button padding: `12px` vertical, `16px` horizontal

## Layout
- Mobile-first
- Max content width: `640px`
- Single-column forms
- Full-width buttons on mobile

## Form Patterns
- Use accessible label + input pairs
- Put error text below form fields
- Include a summary/review before payment

## Colour Usage
- Buttons in primary colour (`#1F3A2E`)
- Hover states slightly darker
- Disabled states use muted neutral (`#6B7280`)

## Components
- Input fields: `8px` radius, border-gray
- Buttons: full-width on mobile
- Cards: subtle border, rounded
