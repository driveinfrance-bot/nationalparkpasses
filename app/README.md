# Crit’Air Permit Application Site

This project is a web application for a private service that helps English-speaking drivers apply for the France Crit’Air emissions permit.

The site provides a clear, English-friendly explanation of Crit’Air requirements and a simple application flow to collect vehicle details, upload registration documents, and process payments.

## Key Features
- Marketing landing page explaining Crit’Air in plain English
- Multi-step application flow for vehicle details and document upload
- Stripe payments (card, Apple Pay, Google Pay)
- Blog section for SEO content
- Clear disclosure that this is a private service and not affiliated with the French government

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Stripe
- Supabase (Postgres + Storage)

## Data & Privacy
- Minimal personal data collected (vehicle details + email)
- Uploaded documents used only for permit processing
- Data deleted after fulfilment according to the privacy policy
